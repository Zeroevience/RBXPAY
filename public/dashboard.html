API Documentation
Integrate RBXPAY into your game or website
Base URL
http://localhost:3000
Create Payment
POST /api/payments/create — Initiates a new payment session and returns a payment URL.
curl -X POST http://localhost:3000/api/payments/create \ -H "Content-Type: application/json" \ -d '{ "apiKey": "rbxpay_your_key_here", "gampassUrl": "https://www.roblox.com/game-pass/12345/My-Pass", "returnUrl": "https://yoursite.com/success", "metadata": {"userId": "abc123"} }'
Check Payment Status
GET /api/payments/:paymentId/status — Poll to check if a payment has been completed.
curl http://localhost:3000/api/payments/{paymentId}/status # Response: { "status": "completed" // or "pending" }
Auto Create Gamepass
POST /api/gamepasses/create — Create a gamepass using your linked account cookie.
curl -X POST http://localhost:3000/api/gamepasses/create \ -H "Content-Type: application/json" \ -d '{ "apiKey": "rbxpay_your_key_here", "universeId": "123456789", "name": "VIP Pass", "price": 100, "description": "My gamepass" }'
Payment Flow
The complete checkout flow for your customers — entirely handled by RBXPAY.
1. Create payment → get paymentUrl 2. Redirect user to paymentUrl 3. User completes 7-step checkout 4. Poll /status until "completed" 5. Redirect user to returnUrl (or receive webhook callback)
